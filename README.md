# spring-boot-forecast-seed
A starter project integration with http://openweathermap.org/api 

### Stack:
- Spring Boot
- AngularJS
- Bootstrap
- H2 Database

### Require
- JDK 8
- Bower
- Maven

### Run
```sh
$ bower install
$ mvn spring-boot:run
```

### Integration
- [Travis CI](https://travis-ci.org/heidiks/spring-boot-forecast-seed)
